# July 2026 Device Deals & S-Coin Catalog

This folder is the complete static website. `index.html` is the main page.

## Publish with GitHub Pages

1. Create a new GitHub repository.
2. Upload **all files and folders from this directory** to the repository root. Keep `index.html`, `assets`, `phones`, `scoin-logo.png`, and `.nojekyll` together.
3. In the repository, open **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/ (root)`, then save.
6. GitHub will show the public website address after deployment finishes.

The category links use URL hashes, so the catalog works from a project subdirectory such as `https://username.github.io/repository-name/`.

External product images require an internet connection. Google Pixel and S-Coin assets are included locally.
